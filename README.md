# 🚀 LaunchDeck

LaunchDeck is a modern full-stack project showcase platform where developers can discover, publish, and manage software projects. It provides an intuitive interface for exploring projects with advanced filtering while allowing authenticated users to contribute and manage their own submissions.

## 🌐 Live Demo

- **Client:** https://launch-deck-fawn.vercel.app
- **Server:** https://launchdeck-web-api-server.onrender.com

---

## ✨ Features

- 🔐 Secure authentication using Better Auth
- 📂 Browse all projects with search, filtering, sorting, and pagination
- ⭐ Featured projects showcased on the homepage
- 📄 Detailed project information page
- ➕ Add new projects (protected route)
- 🛠️ Manage personal projects from a dedicated dashboard
- 📱 Fully responsive design for desktop and mobile
- 🎨 Smooth animations and modern UI
- 🔔 Toast notifications for user feedback

## 🚀 Getting Started

### Clone the repository

```bash
git clone <https://github.com/alfaazahmed7/launchDeck-client>
cd launchdeck-client
```

### Install dependencies

```bash
npm install
```

### Create a `.env.local` file

```env
NEXT_PUBLIC_SERVER_URL=your_backend_url
NEXT_PUBLIC_BETTER_AUTH_URL=your_auth_url
```

### Run the development server

```bash
npm run dev
```
